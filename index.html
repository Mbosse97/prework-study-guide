HTML

  • The head element contains information about the webpage.
  • The head element also contains the metadata for the webpage
  • The body element represents the visible content shown to the user.
  • semantic elements are elements which describe the use of the tag content

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • color will apply to the color of the text but background color is the backgroundo of the element
  • inline css can be used within the html document itself

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git init will create a hidden git file within the folder to make it a local repository
  • git push will add your commits to the cloud based repository from your local system

JavaScript

  • For loops can be used to iterate over an array one index at a time
  • An if statement acts as a conditional function to initiate an action once a specific action has been made
  • Javascript is used to add functionality to the website
  • Arrays can be used to store multiple pieces of information about a variable
